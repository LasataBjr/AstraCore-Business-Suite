# AstraCore Business Suite

AstraCore Business Suite is a full-stack Laravel 11 content and business management system designed for modern agencies, startups, and service-based companies. It provides a dynamic frontend website and a powerful admin dashboard for managing business content without modifying source code.

---

## 📖 Overview

AstraCore Business Suite allows administrators to manage:

- Services
- Projects
- Team Members
- Testimonials
- Blog Posts
- Contact Inquiries
- Website Settings

The platform combines a professional public-facing website with a secure administration panel built using Laravel 11.
